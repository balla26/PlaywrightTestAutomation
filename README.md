# Playwright Test Automation

## Precondition
- Node.js 14+ must be installed.

## Clone the Repository
```sh
git clone https://github.com/balla26/PlaywrightTestAutomation.git
```

## Install Dependencies
```sh
npm install
```

## Execute Test
```sh
npx playwright test ./tests/filename
```
